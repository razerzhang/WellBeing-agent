# 🚀 快速启动指南

## 1. 安装依赖

```bash
pip install -r requirements.txt
```

## 2. 配置环境变量

```bash
cp env.example .env
# 编辑 .env 文件，添加你的API密钥
```

## 3. 启动后端服务器

```bash
python start_server.py
```

## 4. 访问前端

在浏览器中打开：http://localhost:8000

## 5. 开始使用

- 输入健康相关问题
- 点击快速咨询按钮
- 享受AI健康顾问服务
